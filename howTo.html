How to get the app?

To get the app, contact developers.

How to install?

Contact the developer for apk file and install it.Note that while installing you might get prompted with a message of "Allow Installation from unknown sources", simply allow it.

How to use?

1.Pick me

To send a request to selected bus driver to pick you at the bus stop nearest to you.

2. Search nearest bus

To find the bus nearest to you among all the buses.

3. Choose bus

You can choose a bus to see its current location.

4. Distance

It displays the distance between you and the selected bus.

5. Refresh

It is used to refresh the applicatiton

System Requirements

Android version of phone should be kitkat or higher.
