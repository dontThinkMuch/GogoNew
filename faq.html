1. Is my password secure?

We are using firebase authentication service provided by Google which ensures that your password is encrypted and stored safely.

2. How accurate is the distance and time calculated by the app?

Initially, when you select a bus distance and time shown are provided by google api. Later on and till the time you select another bus, distance and time are calculated using Haversine formula.

3. Why is google api not being used all the time for calculating distance and time?

We are restricted to use google api request limited to 1000 per day. So, we can't use google api all the time.
